# numberFormatJs
 javascript version of the number_format function in php



## Usage

```javascript
number_format(1.5) // returns '1'
``` 

```javascript
number_format(1.5, 2) // returns '1.50'
``` 

```javascript
number_format(1.5, 2, ',') // returns '1,50'
``` 

```javascript
number_format(11458.3547, 3, ',', ".") // returns '11.458,354'
```
